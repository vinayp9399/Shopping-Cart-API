To run this project locally:  
  
1. Install requirements:  
  
```console  
$ pip install -r requirements.txt  
```  
  
2. Set up DB:   
   
```console  
$ python manage.py makemigrations   
$ python manage.py migrate   
```  
   
3. Run the app:   
  
```console   
$ python manage.py runserver  
```
